SQL2GEOJSON API Server

If DB = 'postgres'

  • API is available on '/postgres/api/<insert-table-name-here>
  • Filtering: '/postgres/api/<insert-table-name-here>?fields=<field-1-you-want-to-filter>...&filter=<filter-term>' 
    Minor Bug: Combining fields with different data types and string filter results in error.
See https://github.com/sudiptog81/sql2geojson/issues/1 for updates on this bug.
  • For e.g. /postgres/api/test and /postgres/api/test?fields=dev&filter=XYZ

If DB = 'mysql'

  • API is available on '/mysql/api/<insert-table-name-here>?fields=<field-1-you-want>,<field-2-you-want>,...'
  • Filtering: '/mysql/api/<insert-table-name-here>?fields=<field-1-you-want>,<field-2-you-want>...&filter=<filter-term>'
  • For e.g. /mysql/api/test?fields=id,name,dev and filtered: /mysql/api/test?fields=id,name,dev&filter=XYZ

To view an example go to /example after importing the shapefiles (available here) into your database and make sure
that the SQL2GEOJSON API Server is running.
